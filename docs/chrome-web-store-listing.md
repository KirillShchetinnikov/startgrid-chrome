Localized copy for every language currently supported by StartGrid.

## English (`en`)

### Summary

A customizable new tab with visual bookmarks, folders, search, thumbnails, themes, and optional Chrome Sync.

### Description

StartGrid turns every new tab into a clean, customizable space for your bookmarks.

Keep using your existing Chrome bookmarks and organize them in a familiar folder tree. Create, edit, delete, and rearrange bookmarks and folders with drag and drop. Choose a default folder, quickly search your bookmarks, or search the web with your preferred search engines.

Make the page your own:

• Choose website screenshots, image URLs, local images, or favicons for bookmark tiles
• Adjust the grid, tile size, titles, favicons, sorting, and folder previews
• Use a light, dark, or system theme
• Set a local image or video, an external image, or Bing Image of the Day as the background
• Configure built-in and custom search engines
• Assign keyboard shortcuts to common actions

StartGrid also lets you import and export extension settings. Optional Chrome Sync can keep supported settings consistent across your Chrome profiles. Bookmark thumbnails and other local images stay on the device.

No StartGrid account is required. The extension works with Chrome's bookmark system and remains useful even when optional permissions or synchronization are disabled.

Выберите папку по умолчанию, быстро находите нужные закладки или ищите в интернете с помощью удобных вам поисковых систем.

Настройте страницу под себя:

• Используйте для плиток снимки сайтов, изображения по URL, локальные картинки или favicon
• Настройте сетку, размер плиток, подписи, favicon, сортировку и превью папок
• Выберите светлую, тёмную или системную тему

Учётная запись StartGrid не требуется. Расширение работает со стандартной системой закладок Chrome и сохраняет основные возможности даже без дополнительных разрешений и синхронизации.
